Este es un repositorio que contiene funciones JS para utilizar en diversos proyectos

Indice de funciones:
---------------------------

formatDashNum => formatea un numero de 10 digitos mientras se ingresa a un numero con formato 000-000-0000
formatDateSlash => formatea fecha. formato salida dd/mm/yyyy 
deleteOneItem => recibe un array y un elemento => devuelve el array sin el elemento
